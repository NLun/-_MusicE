# 音之萃MusicE


### 簡介  
這是一個結合樂器分離系統「Spleeter」以及自動音樂轉錄系統「Omnizart」，所設計的GUI應用軟體，希望任何背景的人都可以透過它來輕鬆協助音樂工作。  

### 功能：  
導入任一混合音樂，從「鋼琴」、「歌聲」、「鼓組」當中選擇輸出的樂器，系統將分離並轉錄出所選樂器之MIDI檔，供下載。
